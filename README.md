# HidrometeorologíaChile – Frontend MVP

Este repositorio contiene el frontend MVP del proyecto **HidrometeorologíaChile**, desarrollado utilizando **React + Vite**.

La aplicación corresponde a una maqueta funcional enfocada en la consulta, visualización, filtrado y comparación de información hidrometeorológica proveniente de distintas estaciones meteorológicas.

## Requisitos

Antes de ejecutar el proyecto, es necesario tener instalado:

- Node.js
- Git

## Repositorio

```bash
https://github.com/vicentekrebs/GPTI
```

## Cómo ejecutar la aplicación

Clonar el repositorio:

```bash
git clone https://github.com/vicentekrebs/GPTI.git
```

Ingresar a la carpeta del proyecto:

```bash
cd GPTI
```

Instalar dependencias:

```bash
npm install
```

Ejecutar la aplicación:

```bash
npm run dev
```

Luego abrir en el navegador el link local que aparecerá en la terminal (normalmente `http://localhost:5173`).

## Posible error con Recharts

Si aparece un error relacionado con la librería `Recharts`, ejecutar:

```bash
npm install recharts
npm run dev
```
